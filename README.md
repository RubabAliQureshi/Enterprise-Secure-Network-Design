# 🌐 Enterprise Secure Network Design

A full enterprise network architecture designed for a multi-department organization requiring segmented access, dynamic routing, and secure remote connectivity — achieving zero unintended cross-VLAN access.

## 🎯 Key Results
- ✅ **Zero unintended cross-VLAN access** in simulation
- ✅ **100% VPN uptime** across all tested failover scenarios
- ✅ Full department isolation achieved

## 🔐 Security Features
- **VLAN Segmentation** for department isolation
- **OSPF Dynamic Routing** for intelligent path selection
- **NAT & ACLs** for traffic filtering and control
- **Site-to-Site VPN** for secure remote connectivity
- Inter-VLAN access policies enforced at distribution layer
